Sobre W&DB

"Writed & Directed By" es el proyecto de final de grado de Juan Carlos Fernández González.

El objetivo de este proyecto es el desarrollo al completo de un directorio en red para el amacenado y visualización de películas
todo administrado por un sistema de login y register a través de una base de datos.
El proyecto surje de las ganas que tenía de desarrollar un directorio donde pudiese tener mi propia versión de Netflix, o HBO, por lo que
con tiempo, quebraderos, tutoriales y muucho ChatGPT he desarrollado un ecosistema completo bastante competente y los más importante
¡CON 0 PLANTILLAS!
Esta estructura de servicio consta de dos Servidores:

- Servidor Web
Donde estás ahora mismo leyendo esto, una máquina Ubuntu Server 22.04 con Apache2 (2.4.52) el cual tiene montada una estructura completa
de directorios donde se almacenan todos los ficheros necesarios. Este proyecto usa para el FrontEnd: HTML5, CSS, JS y PHP, este último es
necesario para la conexión con el segundo servidor, que es el...

- Servidor BBDD
Donde se consultan, procesan y almacenan todos los datos de usuarios y/o películas del entorno, este servidor también está montado sobre Ubuntu Server 22.04
el cual tiene instalada una Base de Datos MySQL, con la que, haciendo uso de dos tablas llamadas usuarios y peliculas, se consulta y administra
todo lo relacionado con el tratamiento de datos ya sea de las películas o de los usuarios que se registren y accedan a la plataforma.
Toda la información y código de este proyecto puede ser consultado en mi GitHub.
