Información Sobre W&DB

"Writed & Directed By" es el proyecto de final de grado de Juan Carlos Fernández González.

El objetivo de este proyecto es el desarrollo al completo de un ecosistema en red para el amacenado y visualización de películas. El proyecto surje de las ganas que llevaba queriendo desarrollar algo asi, por lo que, con cuestión de tiempo, quebraderos, tutoriales y muucho ChatGPT estoy consiguiendo desarrollar un ecosistema completo bastante competente, esta estructura de servicio consta de dos partes bien diferenciadas, un servidor Web, que es donde estás ahora mismo leyendo esto y un servidor de Bases de Datos, este primer servidor Web está basado en Ubuntu Server 22.04 el cual tiene montado un servidor Apache2, la estética está desarrollada con HTML, CSS y JavaScript, lo cual no ha sido fácil ya que nunca en mi vida habia hecho nada relacionado con JS, aunque haciendo uso del práctico ensayo/error he conseguido que quede un FrontEnd bastante competente.

Así que veremos que rumbo coje el proyecto, por ahora hasta aquí puedo leer.
